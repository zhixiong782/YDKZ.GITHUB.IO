# 行星减速机

![行星减速机](https://images.unsplash.com/photo-1504917595217-d4dc5ebe6122?w=600)

## 产品概述

行星减速机采用精密的行星齿轮传动设计，具有高精度、高扭矩输出、低噪音等特点，是提高运动系统性能的理想传动装置。

## 主要特点

- **高精度**：背隙小于3弧分
- **高扭矩密度**：紧凑设计，大扭矩输出
- **低噪音**：精密磨齿，运行平稳
- **高效率**：单级效率95%以上
- **长寿命**：优质材料和精密制造

## 技术参数

| 参数项 | 参数值 |
|--------|--------|
| 减速比 | 3-100 |
| 额定扭矩 | 10-5000Nm |
| 最大扭矩 | 20-10000Nm |
| 背隙 | <3弧分（精密型） |
| 效率 | ≥95% |
| 防护等级 | IP64 |
